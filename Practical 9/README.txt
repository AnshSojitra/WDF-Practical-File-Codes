Title: Form Submission and Data Storage using PHP

Objective:
To collect form data using PHP and store it in different formats (TXT, CSV, JSON).

Tools Used:
- PHP
- XAMPP (Apache + PHP runtime)

Implementation Time: 4 hours
Engagement Level: 6/10

Files:
- form.html: HTML form for user input
- submit.php: Stores data in plain text file
- submit_csv.php: Stores data in CSV format
- submit_json.php: Stores data in JSON format
- display_json.php: Displays JSON data dynamically
- data.txt / data.csv / data.json: Output files

Key Features:
- Uses POST method for form submission ✅
- Input validation and sanitization using PHP ✅
- Confirmation message displayed after submission ✅
- Intermediate extension: CSV storage ✅
- Advanced extension: JSON storage and dynamic display ✅

Evaluation Strategy:
- Demonstrate form submission flow
- Show contents of output files
- Explain code logic and file handling

Learning Outcomes:
- Understand PHP form handling
- Learn file writing in different formats
- Apply input sanitization and validation
- Display structured data dynamically

COs Mapped:
- CO1: Apply basic PHP for form handling
- CO5: Implement structured data storage and dynamic display
